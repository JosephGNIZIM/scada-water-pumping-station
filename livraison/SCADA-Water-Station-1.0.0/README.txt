SCADA Water Station 1.0.0

Contenu
- SCADA Water Station 1.0.0.exe

Installation / lancement
1. Copier ce dossier sur le PC cible.
2. Double-cliquer sur "SCADA Water Station 1.0.0.exe".
3. Si Windows affiche SmartScreen:
   - cliquer sur "Informations complementaires"
   - puis sur "Executer quand meme"

Prerequis
- Windows 10 ou Windows 11 64 bits

Notes
- Cette version est portable: aucune installation classique n'est requise.
- Au premier lancement, l'application peut prendre quelques secondes a demarrer.
- Si Mosquitto n'est pas installe, l'application reste utilisable, mais le demarrage automatique du broker MQTT local est desactive.

Support
- En cas de blocage, verifier que le pare-feu Windows n'empeche pas l'application de lancer ses services locaux.
